## 2. SS に入れた機能（このセッション）

- **Googleカレンダー双方向連携**（GIS / Calendar API v3）。差分取り込み(syncToken)・編集はデバウンスでまとめ送信・**自動接続**（開くと無音で接続→同期、トークンはメモリ＋localStorageキャッシュ、ログインヒントでアカウント選択スキップ）。
- **イベントの通知オフ**（reminders 無効）。既存にも効く「全件再送信」ボタンあり。
- **複数日（期間）予定**＝`endDate` 追加。カレンダーに各日展開（`n/N` 表示）。
- **Undo / Redo**（⌘Z・Ctrl+Z／⌘⇧Z・Ctrl+Y、最大100手）。
- **隔週など繰り返し予定の「この回だけ削除／繰り返し全部削除」**。この回だけはGoogle側のその回（インスタンス）も削除。
- **カレンダー月曜始まり**＋土日セルの薄い色付け（日=赤／土=青）。
- **横幅をウィンドウ可変**（max 1600px）。カレンダー7列の見切れ修正。
- **予定の一括取り込み（JSONインポート）**（設定タブ）。
- **ショート進捗の段階表示**：台本→撮影済→ストック→投稿済＋上部にステージ集計バー。
- アイコンv2（山モチーフ）適用、UIミニマル上質化、山の背景（すりガラス）、フォント（Outfit＋Zen Kaku Gothic New）。

## 3. 週次スケジュール設計（隔週ローテ）

配信は基本 **21:00〜23:59**。アンカーは A週=休み週、B週=ガバナイ稼働週。

**B週（ガバナイ稼働・6/29〜7/5起点）**
- 月 配信（RTA/ゲーム）／火 編集／水 配信（個人ゲーム）※追加／木 ガバナイ ゲームデー／金 編集／土 雑談配信／日 ガバナイ ラジオ
- ガバナイ週は配信が全部夜なので**土日の昼は編集に使える**（昼=編集／夜=配信の二毛作）。個人配信＝月・水・土の3本。

**A週（週末休み・7/6〜7/12起点）**
- 月 編集／火 雑談配信／水 編集／木 配信（RTA/ゲーム）／金・土・日 休み
- 「木配信→金土日休み→月配信」の流れを維持。

スケジュールは `summitsirdar_schedule_plan.json`（本リポジトリ）を設定タブの一括取り込みに貼って登録。

## 4. コンテンツ戦略の決定事項

- **作業配信**：編集は配信外。Live2D作業は9月末納品後（10月〜）。新衣装は内密＝作業配信しない（リビールはイベント化して別途仕込む）。当面の作業配信は「画面なしのもくもく雑談」枠のみ可。
- **雑談 vs 動画投稿**：雑談は「深さ（固定深耕）」、動画投稿（ショート）は「広さ（新規発見）」。目標がリーチ拡大なので**伸ばすのは動画投稿**。雑談は週1で据え置き。ショート量はボトルネック＝編集時間なので、**VOD切り抜き系（軽編集）で本数を増やす**。当たり型を見つけてから量産。
- **ショート**：まず5本ストック。1技を切り口違い（衝撃/解説/コツ）でA/Bテスト。台本仕様・初動ラインナップは CLAUDE.md 参照。
